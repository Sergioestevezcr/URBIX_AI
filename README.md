# 🚦 URBIX AI

### Smart Mobility · Artificial Intelligence · GovTech

URBIX AI es un proyecto académico desarrollado en **Proyecto Integrador I** de Ingeniería de Sistemas en la **Escuela Colombiana de Ingeniería Julio Garavito**.

La propuesta plantea una solución conceptual enfocada en **Smart Cities** y optimización inteligente del tráfico urbano mediante inteligencia artificial, visión computacional y analítica de datos en tiempo real.

---

## 🌍 Problemática

Bogotá enfrenta altos niveles de congestión vehicular, afectando:

* Calidad de vida
* Salud mental y estrés vial
* Productividad urbana
* Movilidad de emergencias
* Emisiones de CO₂

URBIX AI nace como una propuesta tecnológica orientada a repensar la movilidad urbana desde un enfoque inteligente y sostenible.

---

## 🧠 ¿Qué es URBIX AI?

URBIX AI es una plataforma conceptual de optimización semafórica basada en inteligencia artificial.

El proyecto propone una arquitectura capaz de:

* Analizar patrones de tráfico
* Optimizar tiempos semafóricos
* Visualizar datos urbanos
* Priorizar vehículos de emergencia
* Mejorar la movilidad mediante analítica predictiva

## 🌐 Acceso al sitio web

Debido a que el proyecto está alojado en un servidor académico de la Escuela Colombiana de Ingeniería Julio Garavito, algunos navegadores pueden mostrar una advertencia de conexión no segura al ingresar al enlace.

Esto ocurre porque el certificado SSL del servidor institucional no está validado globalmente, pero el sitio sigue siendo accesible.

### Pasos para ingresar:

1. Abrir el enlace del proyecto:

   https://estudiantes.is.escuelaing.edu.co/~1000114329/urbix_ai/index.html

2. Cuando aparezca la advertencia:

   **“Your connection is not private”**

3. Seleccionar:

   **Advanced**

4. Luego hacer clic en:

   **Proceed to estudiantes.is.escuelaing.edu.co (unsafe)**

5. El sitio cargará normalmente.

> Nota: El proyecto es únicamente una página web informativa académica y no solicita datos personales, contraseñas ni información sensible.
